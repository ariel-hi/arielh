# Ariel H — Personal Website

A collection of interactive pages, games, and visual experiments.

## Pages

| Page | Description |
|------|-------------|
| **Home** | Kinetic canvas landing page |
| **Rufus** | Interactive photo gallery |
| **Horizon** | Visual experiment |
| **Nebula** | Cosmic nebula effect |
| **Shooter** | Space shooter arcade game with leaderboard |
| **Gems** | Star Cluster Blitz — match-style gem game |
| **Grid16** | 16 simultaneous microgames — survive the grid |

## Running Locally

Open `index.html` in a browser, or serve with any static file server:

```bash
npx serve .
```

## Tech

- Vanilla HTML / CSS / JavaScript — no frameworks
- Canvas-based rendering for games and effects
- Supabase for leaderboards (Shooter & Grid16)
